# TopMapSolutions OnSite

TopMapSolutions OnSite is a React Native app that uses MapLibre for mapping features.

1. quick start
Create a new Expo project using the blank template:
```
npx create-expo-app@latest --template blank
npx expo start
```
---

2. Install Expo Router
To enable file-based routing in your project:
```
npm install expo-router
```
Official Router Guide: [Expo Router] (https://docs.expo.dev/versions/latest/sdk/router/)
---

3. Installing Icons
```
npm install -g expo
expo install @expo/vector-icons

```
preview of icons: [Expo Icons](https://icons.expo.fyi/Index)
